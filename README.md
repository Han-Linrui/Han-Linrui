# Linrui Han

I am an undergraduate student at **Shanghai Jiao Tong University (SJTU)**, majoring in Data Science. My academic background combines rigorous mathematical training with strong system programming skills (C++ / Python).

My primary research interest lies in the intersection of **Reinforcement Learning (RL) and Generative Models**. Specifically, I am interested in how optimization algorithms and RL mechanisms (e.g., Bandits, PPO, Test-Time Search) can be utilized to constrain, align, and guide generative priors, ensuring their outputs are logically sound and environmentally grounded.

### 🔬 Featured Project
[**LOLA-RL-Alignment**]((https://github.com/Han-Linrui/LOLA-Algorithm-Reproduction-Analysis)) 
: Mitigating Generative Biases via Online Bandit Feedback
- **Overview**: An algorithmic reproduction and sensitivity analysis of the LLM-Assisted Online Learning Algorithm (LOLA, Marketing Science 2025). 
- **Methodology**: Rather than relying solely on static generative priors, this project formulates the exploration-exploitation trade-off by integrating a Data-Driven UCB with an LLM-Augmented UCB. The algorithm bounds the generative hallucination risk using empirical environmental feedback.
- **Takeaway**: This project shaped my understanding of how RL mechanisms can dynamically correct the distributional biases of large models, motivating my future research in alignment and decision-making for foundation models.
